GOAL

Generate a minimal but production-ready ecommerce optimized for Instagram and WhatsApp sales.

The system must allow a store owner to manage products through an admin panel and customers to buy through Mercado Pago.

IMPORTANT REQUIREMENTS

1. The project must be structured following modern Next.js best practices.

2. Generate all folders and files.

3. The admin panel must include a full CRUD (ABM) for products.

4. Use Supabase as the database provider.

5. Implement a checkout endpoint that creates a Mercado Pago preference.

6. Implement a webhook endpoint to receive payment confirmation.

7. Create reusable components.

8. The project must be easily clonable as a template for multiple stores.
